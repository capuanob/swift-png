Instance Method

read(count:)

Reads the specified number of bytes from this file interface.

Implements requirement in PNG.Bytestream.Source.

Declaration

func read(count capacity:Int) -> [UInt8]?

Parameters

capacity

The number of bytes to read.

Return value

An array containing the read data, or nil if the specified number of bytes could not be read.

Overview

This method only returns an array if the exact number of bytes specified could be read. This method advances the file pointer.
