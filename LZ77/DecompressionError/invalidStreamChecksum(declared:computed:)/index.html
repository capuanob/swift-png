Enumeration Case

invalidStreamChecksum(declared:computed:)

The modular redundancy checksum computed on the uncompressed data did not match the checksum declared in the compressed data stream footer.

Declaration

case invalidStreamChecksum(declared:UInt32, computed:UInt32)

Associated values

declared

The checksum declared in the compressed data stream footer.

computed

The checksum computed on the uncompressed data.

Overview

This error should not be confused with invalidStreamHeaderCheckBits. Nor should it be confused with PNG.LexingError.invalidChunkChecksum(declared:computed:), which refers to the cyclic redundancy checksum in every PNG chunk.
