Instance Method

push(data:overdraw:)

Decompresses the contents of an IDAT chunk, and updates the image state with the newly-decompressed image data.

Declaration

mutating func push(data:[UInt8], overdraw:Bool) throws

Parameters

data

The contents of the IDAT chunk to process.

overdraw

If true, pixels that are not yet available will be filled-in with values from nearby available pixels. This option only has an effect for interlaced images.

The default value is false.
