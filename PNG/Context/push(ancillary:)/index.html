Instance Method

push(ancillary:)

Parses an ancillary chunk appearing after the last IDAT chunk, and adds it to the image metadata.

Declaration

mutating func push(ancillary chunk:(type:Chunk, data:[UInt8])) throws

Parameters

chunk

The chunk to process. Its type must be one of tIME, iTXt, tEXt, zTXt, or IEND, or a private application data chunk type.

All other chunk types will throw appropriate errors.

Overview

This function validates the multiplicity of the given chunk, and its chunk ordering with respect to the IDAT chunks. The caller is expected to have consumed all preceeding IDAT chunks in the image being decoded.

Despite its name, this function can also accept an IEND critical chunk, in which case this function will verify that the compressed image data stream has been properly-terminated.
