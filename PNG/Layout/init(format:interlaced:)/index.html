Initializer

init(format:interlaced:)

Creates an image layout.

Declaration

init(format:Format, interlaced:Bool)

Parameters

format

A color format.

interlaced

Specifies if the image uses interlacing. The default value is false.

Overview

This initializer will validate the fields of the given color format. Passing an invalid format will result in a precondition failure.
