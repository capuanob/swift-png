Structure

PNG.Histogram

A palette frequency histogram.

Declaration

struct Histogram

Overview

This type models the information stored in a hIST chunk.

Topics

Parsing and serialization

init(parsing:[UInt8], palette:Palette) throws

Creates a palette histogram by parsing the given chunk data, validating it according to the given image palette.

var serialized:[UInt8]

Encodes this histogram as the contents of a hIST chunk.

Initializers

init(frequencies:[UInt16], palette:PNG.Palette)

Creates a palette histogram.

Instance properties

let frequencies:[UInt16]

The frequency values of this histogram.
