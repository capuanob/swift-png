Static Method

unpack(_:of:deindexer:)

Unpacks an image data storage buffer to an array of this color target, using a custom deindexing function.

Required.

Declaration

static func unpack(_ interleaved:[UInt8], of format:Format, deindexer:([(r:UInt8, g:UInt8, b:UInt8, a:UInt8)]) -> (Int) -> Aggregate) -> [Self]

Parameters

interleaved

An image data buffer. It is expected to be obtained from the storage property of a Rectangular image.

format

The color format associated with the given data buffer. It is expected to be obtained from the the layout.format property of a Rectangular image.

deindexer

A function which uses the palette entries in the color format to generate a dereferencing function. This function should only be invoked if the color format is an indexed format.

See the indexed color tutorial for more about the semantics of this function.

Return value

A pixel array containing instances of this color target. The pixels should appear in the same order as they do in the image data buffer.
