Enumeration Case

bgra8(palette:fill:)

An 8-bit BGRA color format.

Declaration

case bgra8(palette:[(b:UInt8, g:UInt8, r:UInt8)], fill:(b:UInt8, g:UInt8, r:UInt8)?)

Associated values

palette

An palette of suggested posterization values. Most PNG viewers ignore this field.

This field is unrelated to, and should not be confused with a SuggestedPalette.

fill

An optional background color. Most PNG viewers ignore this field.

Overview

This color format is an iphone-optimized format. It has a pixel format of rgba8.
