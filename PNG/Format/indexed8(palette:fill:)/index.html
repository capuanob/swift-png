Enumeration Case

indexed8(palette:fill:)

An 8-bit indexed color format.

Declaration

case indexed8(palette:[(r:UInt8, g:UInt8, b:UInt8, a:UInt8)], fill:Int?)

Associated values

palette

The palette values referenced by an image with this color format. This palette must be non-empty, and can have at most 256 entries.

fill

A palette index specifying an optional background color. This index must be within the index range of the palette array.

Most PNG viewers ignore this field.

Overview

This color format has a pixel format of indexed8.
