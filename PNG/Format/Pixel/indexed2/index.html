Enumeration Case

indexed2

Pixels are stored as 2-bit indices.

Declaration

case indexed2

Overview

An image with this pixel format has a bit depth of 2, and a color depth of 8. Each index is in the range 0 ... 3, and can reference an entry in a palette with at most 4 elements.
