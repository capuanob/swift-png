Enumeration

PNG.Format.Pixel

A pixel format.

Declaration

enum Pixel

Overview

A pixel format specifies the color model and bit depth used by an image. They do not specify the ordering of the color samples within the internal representation of a PNG image. For example, the color formats rgba8(palette:fill:) and bgra8(palette:fill:) both correspond to the pixel format rgba8.

The pixel format associated with a color format can be accessed through the pixel instance property.

Instance properties
