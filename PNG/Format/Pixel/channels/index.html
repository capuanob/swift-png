Instance Property

channels

The number of channels encoded per-pixel in the internal representation of an image with this pixel format.

Declaration

@inlinable var channels:Int { get }

Overview

This number is not the number of components in the encoded image; it indicates the dimensionality of the stored image data. Notably, indexed images are defined as having one channel, even though each scalar index represents a four-component color value.

This property returns 1 for all grayscale and indexed pixel formats.

This property returns 2 for all grayscale-alpha pixel formats.

This property returns 3 for all RGB pixel formats.

This property returns 4 for all RGBA pixel formats.
