Enumeration Case

rgb8(palette:fill:key:)

An 8-bit RGB color format.

Declaration

case rgb8(palette:[(r:UInt8, g:UInt8, b:UInt8)], fill:(r:UInt8, g:UInt8, b:UInt8)?, key:(r:UInt8, g:UInt8, b:UInt8)?)

Associated values

palette

An palette of suggested posterization values. Most PNG viewers ignore this field.

This field is unrelated to, and should not be confused with a SuggestedPalette.

fill

An optional background color. Most PNG viewers ignore this field.

key

An optional chroma key. If present, pixels matching it will be displayed as transparent, if possible.

Overview

This color format has a pixel format of rgb8.
