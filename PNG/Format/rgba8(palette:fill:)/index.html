Enumeration Case

rgba8(palette:fill:)

An 8-bit RGBA color format.

Declaration

case rgba8(palette:[(r:UInt8, g:UInt8, b:UInt8)], fill:(r:UInt8, g:UInt8, b:UInt8)?)

Associated values

palette

An palette of suggested posterization values. Most PNG viewers ignore this field.

This field is unrelated to, and should not be confused with a SuggestedPalette.

fill

An optional background color. Most PNG viewers ignore this field.

Overview

This color format has a pixel format of rgba8.
