Structure

PNG.SuggestedPalette

A suggested image palette.

Declaration

struct SuggestedPalette

Overview

This type models the information stored in an sPLT chunk. It should not be confused with the suggested palette stored in the color Format of an RGB, BGR, RGBA, or BGRA image.

Topics

Parsing and serialization

init(parsing:[UInt8]) throws

Creates a suggested palette by parsing the given chunk data.

var serialized:[UInt8]

Encodes this suggested palette as the contents of an sPLT chunk.
