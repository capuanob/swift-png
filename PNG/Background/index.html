Structure

PNG.Background

A background descriptor.

Declaration

struct Background

Overview

This type models the information stored in a bKGD chunk. This information is used to populate the fill field in an image color Format.

The value of this descriptor is stored in the case property, after validation.

Topics

Parsing and serialization

init(parsing:[UInt8], pixel:Format.Pixel, palette:Palette?) throws

Creates a background descriptor by parsing the given chunk data, interpreting and validating it according to the given pixel format and image palette.
