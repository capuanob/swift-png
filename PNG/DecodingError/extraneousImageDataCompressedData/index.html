Enumeration Case

extraneousImageDataCompressedData

The decoder encountered additional IDAT chunks after the end of the compressed image data stream.

Declaration

case extraneousImageDataCompressedData

Overview

This error should not be confused with an unexpected(chunk:after:) error with both fields set to IDAT, which indicates a non-contiguous IDAT sequence.
