Enumeration

PNG.DecodingError

A decoding error.

Declaration

enum DecodingError:Error

Topics

Enumeration cases

case required(chunk:Chunk, before:Chunk)

The decoder encountered a chunk of a type that requires a previously encountered chunk of a particular type.

case duplicate(chunk:Chunk)

The decoder encountered multiple instances of a chunk type that can only appear once in a PNG file.

case unexpected(chunk:Chunk, after:Chunk)

The decoder encountered a chunk of a type that is not allowed to appear after a previously encountered chunk of a particular type.

case extraneousImageData

The compressed image data stream produces more uncompressed image data than expected.

case extraneousImageDataCompressedData

The decoder encountered additional IDAT chunks after the end of the compressed image data stream.

case incompleteImageDataCompressedDatastream

The decoder finished processing the last IDAT chunk before the compressed image data stream was properly terminated.
