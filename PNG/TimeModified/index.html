Structure

PNG.TimeModified

An image modification time.

Declaration

struct TimeModified

Overview

This type models the information stored in a tIME chunk. This type is time-zone agnostic, and so all time values are assumed to be in universal time (UTC).

Topics

Parsing and serialization

init(parsing:[UInt8]) throws

Creates an image modification time by parsing the given chunk data.

var serialized:[UInt8]

Encodes this image modification time as the contents of a tIME chunk.
