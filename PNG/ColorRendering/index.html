Enumeration

PNG.ColorRendering

A color rendering mode.

Declaration

enum ColorRendering

Overview

This type models the information stored in an sRGB chunk. It is not recommended for the same image to include both a ColorRendering mode and a ColorProfile.

Topics

Parsing and serialization

init(parsing:[UInt8]) throws

Creates a color rendering mode by parsing the given chunk data.

var serialized:[UInt8]

Encodes this color rendering mode as the contents of an sRGB chunk.

Enumeration cases

case perceptual

The perceptual rendering mode.

case relative

The relative colorimetric rendering mode.

case saturation

The saturation rendering mode.

case absolute

The absolute colorimetric rendering mode.
