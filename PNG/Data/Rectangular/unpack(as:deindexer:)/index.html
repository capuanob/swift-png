Generic Instance Method

unpack(as:deindexer:)

Unpacks this image to a pixel array, using a custom deindexing function.

Declaration

@inlinable func unpack<Color>(as _:Color.Type, deindexer:([(r:UInt8, g:UInt8, b:UInt8, a:UInt8)]) -> (Int) -> Color.Aggregate) -> [Color] where Color:Color

Parameters

_

A color target type. This type provides the unpack(_:of:deindexer:) implementation used to unpack the image data.

deindexer

A function which uses the palette entries in the color format to generate a dereferencing function. This function is only expected to be invoked if layout.format is an indexed format.

See the indexed color tutorial for more about the semantics of this function.

Return value

A pixel array. Its elements are arranged in row-major order. The first pixel in this array corresponds to the top-left corner of the image. Its length is equal to size.x multiplied by size.y.
