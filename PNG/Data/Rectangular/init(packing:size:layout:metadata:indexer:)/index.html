Generic Initializer

init(packing:size:layout:metadata:indexer:)

Creates an image from a pixel array, using a custom indexing function.

Declaration

@inlinable init<Color>(packing pixels:[Color], size:(x:Int, y:Int), layout:Layout, metadata:Metadata, indexer:([(r:UInt8, g:UInt8, b:UInt8, a:UInt8)]) -> (Color.Aggregate) -> Int) where Color:Color

Parameters

pixels

A pixel array. Its elements are arranged in row-major order. The first pixel in this array corresponds to the top-left corner of the image. The Color type provides the pack(_:as:indexer:) implementation used to pack the image data.

The length of this array must match size.x * size.y. Passing an array of the wrong length will result in a precondition failure.

size

The size of the image. Both dimensions must be greater than zero. Passing an invalid image size will result in a precondition failure.

layout

An image layout.

metadata

A metadata structure. The default value is an empty metadata structure.

indexer

A function which uses the palette entries in the color format to generate a referencing function. This function is only expected to be invoked if the image color format is an indexed format.

See the indexed color tutorial for more about the semantics of this function.
