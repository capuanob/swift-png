Instance Method

bindStorage(to:)

Rebinds this image to a compatible layout.

Declaration

func bindStorage(to layout:Layout) -> Self

Parameters

layout

The new image layout.

Both the original color format and the new color format must be of the same enumeration case, though the fields may differ. The exceptions are the indexed color formats, which require the lengths of their palette payloads to match exactly.

Return value

An image with the given layout. This image will share backing storage with the original image until it is copied-on-write.

Overview

This interface can be used to switch image layouts without unpacking to and repacking from a color target array. Rebinding to an incompatible layout will result in a precondition failure.
