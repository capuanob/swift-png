Instance Method

chunk()

Lexes a chunk from this bytestream.

Declaration

mutating func chunk() throws -> (type:PNG.Chunk, data:[UInt8])

Return value

The type identifier, and contents of the lexed chunk. The chunk contents do not include the checksum footer.

Overview

This function reads a chunk, validating its stored checksum for data integrity. It reports end-of-stream by throwing LexingError.truncatedChunkHeader or LexingError.truncatedChunkBody(expected:). To recover on end-of-stream, catch these two error cases.

This function is the inverse of Destination.format(type:data:).
