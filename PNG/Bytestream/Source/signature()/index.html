Instance Method

signature()

Lexes the eight PNG signature bytes from this bytestream.

Declaration

mutating func signature() throws

Overview

This function expects to read the byte sequence [137, 80, 78, 71, 13, 10, 26, 10]. It reports end-of-stream by throwing LexingError.truncatedSignature. To recover on end-of-stream, catch this error case.

This function is the inverse of Destination.signature().
