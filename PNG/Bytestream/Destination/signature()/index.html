Instance Method

signature()

Emits the eight PNG signature bytes into this bytestream.

Declaration

mutating func signature() throws

Overview

This function emits the constant byte sequence [137, 80, 78, 71, 13, 10, 26, 10]. It will throw a FormattingError if it fails to write to the bytestream.

This function is the inverse of Source.signature().
