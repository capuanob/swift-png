Instance Method

format(type:data:)

Emits a chunk into this bytestream.

Declaration

mutating func format(type:PNG.Chunk, data:[UInt8]) throws

Parameters

type

The type identifier of the chunk to emit.

data

The contents of the chunk to emit. It should not include a checksum footer, as this function computes and appends it automatically.

The default value is [].

Overview

This function will compute the checksum for the given chunk contents and format it with the appropriate chunk headers and footers. It will throw a FormattingError if it fails to write to the bytestream.

This function is the inverse of Source.chunk().
