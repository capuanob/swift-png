Initializer

init(compressed:keyword:language:content:)

Creates a text comment.

Declaration

init(compressed:Bool, keyword:(english:String, localized:String), language:[String], content:String)

Parameters

compressed

Indicates if the text is to be stored in compressed or uncompressed form within a PNG file.

keyword

A keyword tag, in english, and possibly a non-english language.

The english keyword must contain only unicode scalars in the ranges "\u{20}" ... "\u{7d}" or "\u{a1}" ... "\u{ff}". Leading, trailing, and consecutive spaces are not allowed. There are no restrictions on the localized keyword, other than that it must not contain any null characters.

Passing invalid keyword strings will result in a precondition failure.

If the text is in english, the localized keyword should be set to the empty string "".

language

An array representing an rfc-1766 language tag, where each element is a language subtag.

Each subtag must be a 1â€“8 character string containing alphabetical ASCII characters only. Passing an invalid language tag array will result in a precondition failure.

If this array is empty, then the language is unspecified.

content

The text content. There are no restrictions on it. It is allowed (but not recommended) to contain null characters.
