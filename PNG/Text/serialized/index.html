Instance Property

serialized

Encodes this text comment as the contents of a iTXt chunk.

Declaration

var serialized:[UInt8] { get }

Overview

This property always emits a unicode iTXt chunk, regardless of the type of the original chunk, if it was parsed from raw chunk data. It is the opinion of the library that the latin-1 chunk types tEXt and zTXt are deprecated.
