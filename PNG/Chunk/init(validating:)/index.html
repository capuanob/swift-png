Initializer

init(validating:)

Creates a chunk type identifier, returning nil if the type code is invalid.

Declaration

init?(validating name:UInt32)

Parameters

name

The chunk type code. Bit 13 must be set. If the type code is not a public PNG chunk type code, then bit 29 must be clear.

Overview

This initializer is a non-trapping version of init(name:).
