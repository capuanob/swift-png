Initializer

init(case:pixel:palette:)

Creates a transparency descriptor.

Declaration

init(case:Case, pixel:Format.Pixel, palette:PNG.Palette?)

Parameters

case

A transparency descriptor value.

pixel

The pixel format of the image this transparency descriptor is to be used for. Passing a mismatched enumeration case will result in a precondition failure.

palette

The palette of the image this transparency descriptor is to be used for.

If case is not a palette(alpha:) case, palette must be nil, and the chroma key payload samples must fall within the appropriate range, as determined by the image color depth.

Otherwise, the array payload of the given case must have the same number, or fewer, elements as entries in the given palette.

Overview

This initializer validates the transparency information against the given pixel format and image palette.
