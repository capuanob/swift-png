Enumeration Case

palette(alpha:)

A transparency descriptor for an indexed image.

Declaration

case palette(alpha:[UInt8])

Associated values

alpha

An array of alpha samples, where each sample augments an RGB triple in an image Palette. This array can contain no more elements than entries in the image palette, but it can contain fewer.

It is acceptable (though pointless) for the alpha array to be empty.
