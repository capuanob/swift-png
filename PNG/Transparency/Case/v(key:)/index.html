Enumeration Case

v(key:)

A transparency descriptor for a grayscale image.

Declaration

case v(key:UInt16)

Associated values

key

A chroma key used to display transparency. Pixels matching this key will be displayed as transparent, if possible.

Note that the chroma key is an unscaled sample. If the image color depth is less than 16, only the least-significant bits are inhabited.
