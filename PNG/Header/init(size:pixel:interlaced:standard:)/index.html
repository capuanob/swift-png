Initializer

init(size:pixel:interlaced:standard:)

Creates an image header.

Declaration

init(size:(x:Int, y:Int), pixel:Format.Pixel, interlaced:Bool, standard:Standard)

Parameters

size

An image size, measured in pixels.

Passing a size with a zero or negative dimension will result in a precondition failure.

pixel

A pixel format.

interlaced

Indicates if interlacing is enabled.

standard

Specifies if the header is for a standard image, or an iphone-optimized image.

If standard is ios, then the pixel format must be either rgb8 or rgba8. Otherwise, this initializer will suffer a precondition failure.

Overview

This initializer validates the image size, and validates the pixel format against the given PNG standard.
