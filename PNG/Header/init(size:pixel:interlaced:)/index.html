Initializer

init(size:pixel:interlaced:)

Creates an image header.

Declaration

init(size:(x:Int, y:Int), pixel:Format.Pixel, interlaced:Bool)

Parameters

size

An image size, measured in pixels.

Passing a size with a zero or negative dimension will result in a precondition failure.

pixel

A pixel format.

interlaced

Indicates if interlacing is enabled.
