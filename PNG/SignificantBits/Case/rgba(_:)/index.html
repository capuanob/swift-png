Enumeration Case

rgba(_:)

A color precision descriptor for an RGBA or BGRA image.

Declaration

case rgba((r:Int, g:Int, b:Int, a:Int))

Associated values

_

The number of significant bits in each red, green, blue, and alpha sample, respectively.

The meaning of a color precision descriptor is poorly-defined for BGRA images. It is strongly recommended that iphone-optimized images use SignificantBits only if all samples have the same precision.

Each precision value must be greater than zero, and none of them can be greater than the color depth of the image color format.
