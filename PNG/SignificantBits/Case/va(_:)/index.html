Enumeration Case

va(_:)

A color precision descriptor for a grayscale-alpha image.

Declaration

case va((v:Int, a:Int))

Associated values

_

The number of significant bits in each grayscale and alpha sample, respectively.

Both precision values must be greater than zero, and neither can be greater than the color depth of the image color format.
