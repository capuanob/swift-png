Initializer

init(case:pixel:)

Creates a color precision descriptor.

Declaration

init(case:Case, pixel:Format.Pixel)

Parameters

case

A color precision case. Each precision value in the enumeration payload must be greater than zero, and none of them can be greater than the color depth of the image color format.

pixel

The pixel format of the image this color precision descriptor is to be used for. Passing a mismatched enumeration case will result in a precondition failure.

Overview

This initializer validates the precision information against the given pixel format.
