Structure

PNG.Palette

An image palette.

Declaration

struct Palette:RandomAccessCollection

Overview

This type models the information stored in a PLTE chunk. This information is used to populate the non-alpha components of the palette field in an image color Format, when appropriate.

Topics

Parsing and serialization

init(parsing:[UInt8], pixel:Format.Pixel) throws

Creates an image palette by parsing the given chunk data, interpreting and validating it according to the given pixel format.

var serialized:[UInt8]

Encodes this image palette as the contents of a PLTE chunk.
