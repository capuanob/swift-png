Initializer

init(name:profile:)

Creates a color profile.

Declaration

init(name:String, profile:[UInt8])

Parameters

name

The profile name.

This string must contain only unicode scalars in the ranges "\u{20}" ... "\u{7d}" or "\u{a1}" ... "\u{ff}". Leading, trailing, and consecutive spaces are not allowed. Passing an invalid string will result in a precondition failure.

profile

The uncompressed ICC color profile data. The data will be compressed when this color profile is serialized into an iCCP chunk.
