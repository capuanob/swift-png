pack(_:as:indexer:)

Packs an array of RGBA pixels to an image data storage buffer.

Implements requirement in Color.

Parameters

pixels

An array of RGBA pixels.

format

The color format to pack the given pixels as in the returned data buffer.

When the library uses an implementation of this function to construct a Rectangular image, this color format will be stored in its layout.format property.

indexer

A function which uses the palette entries in the color format to generate a referencing function. This function will only be invoked if the color format is an indexed format. Its palette aggregates will be interpreted as (red, green, blue, alpha) quadruplets.

See the indexed color tutorial for more about the semantics of this function.

Return value

An image data buffer. The packed samples in this buffer appear in the same order as the pixels in the pixels array. (But not necessarily in the same order within each individual pixel.)

When the library uses an implementation of this function to construct a Rectangular image, this data buffer will be stored in its storage property.

Overview

For a grayscale color format, this function selects the r component of each RGBA pixel.

For a grayscale-alpha color format, this function selects the r and a components of each RGBA pixel.

For an RGB or BGR color format, this function selects the r, g, and b components of each RGBA pixel.

The components in each RGBA pixel are assumed to fill the entire range of T.
