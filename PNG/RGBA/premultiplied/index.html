Instance Property

premultiplied

The color obtained by premultiplying the red, green, and blue components of this color with its alpha channel.

Declaration

@inlinable var premultiplied:Self { get }

Overview

The premultiplied color is obtained by invoking premultiply(_:alpha:) on r, g, and b.
