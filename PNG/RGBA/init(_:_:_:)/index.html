Initializer

init(_:_:_:)

Creates an opaque RGBA color.

Declaration

@inlinable init(_ red:T, _ green:T, _ blue:T)

Parameters

red

A red value.

green

A green value.

blue

A blue value.

Overview

The r, g, and b components will be set to red, green, and blue, respectively. The a component will be set to T.max.
