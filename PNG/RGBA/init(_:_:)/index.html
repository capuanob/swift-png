Initializer

init(_:_:)

Creates a monochromatic RGBA color.

Declaration

@inlinable init(_ value:T, _ alpha:T)

Parameters

value

A gray value.

alpha

An alpha value.

Overview

The r, g, and b components will be set to value, and the a component will be set to alpha.
